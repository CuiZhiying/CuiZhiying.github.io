终极错误

好吧，这是一个能反应C指针本质的错误。这种写法，编译没有报错，运行时会出现内存错误。
上面第一行只声明了一个指针变量，里面保存的是一个（32位）的内存地址，但是第二行并没有声明该指针变量保存的是哪一块内存的地址，就直接间接取值符号*来进行初始化了，这样子会让那个数字1无处安放，因为指针变量a还没有指向一块内存。
